# Evgen Media — Лендинг

## Запуск

```bash
pnpm install
pnpm dev
```

Билд:
```bash
pnpm build
```
Статика соберётся в `dist/`.

## Редактирование без кода

Все тексты, кейсы, тарифы и контакты — в файле:

**`app/data/content.ts`**

Открой его и меняй только текст внутри кавычек. Не трогай скобки и запятые вокруг.

- `agencyName` — название агентства
- `heroContent` — заголовок, подзаголовок, кнопка
- `services` — услуги + примеры (image / video / url)
- `cases` — кейсы клиентов
- `pricing` — пакеты. `recommended: true` добавляет бейдж «Выгодно»
- `socials`, `contacts` — ссылки, почта, телефон

## Цвета

Акцентный красный: `#e60012`. Если нужно поменять — найди и замени во всех файлах папки `styles/` и в `globals.css`.

## Добавление кейса/услуги

В `content.ts` найди массив `[...]`, скопируй блок с запятой, вставь новый перед закрывающей `]`.

## Деплой

Загрузи содержимое `dist/` на любой хостинг (Vercel, Netlify, GitHub Pages, Beget и т.д.).
